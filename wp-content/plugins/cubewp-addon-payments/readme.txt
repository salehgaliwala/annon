== Description ==

Once you have built a web app the next best thing to do is to add eCommerce functionality to start monetizing. In order to do so you will need WooCommerce and CubeWP Payment.

Create unlimited pricing plans including Pay per Post or package.

Set a duration for the plan to expire. You can choose between month, year, lifetime or custom prefence.

Monetize with Ad Campaigns by letting users promote their existing posts.

Support all WooCommerce Payment Gateways
